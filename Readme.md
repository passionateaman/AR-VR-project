##  Quick Start

### AR Smart Textbook
1. Download and install APK on Android
2. Grant camera permissions
3. Print image targets from `/AR_SmartTextbook/Assets/ImageTargets/`
4. Point camera at targets to see 3D models

### VR Shooting Game
1. Download APK
2. Install via [SideQuest](https://sidequestvr.com/)
3. Find in "Unknown Sources" on Quest
4. Grab gun with grip button, shoot with trigger

##  Features

**AR Smart Textbook:**
- Real-time image recognition (Vuforia)
- 3D model visualization
- Audio pronunciation
- Objects: Apple, Ball, Cat, Dog

**VR Shooting Game:**
- Full hand tracking
- Physics-based shooting
- Target destruction
- Real-time scoring
